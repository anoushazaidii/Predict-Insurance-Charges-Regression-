# Medical Insurance Cost Prediction (Regression)🚀
Project Overview: This project uses Machine Learning to predict medical insurance costs based on patient data.
I achieved an R² score of 0.8726 by moving from a simple linear baseline to an optimized Random Forest Regressor with custom feature engineering.  
📊 Key Achievements: 
Data Exploration: Analyzed correlations, identifying 'Smoking' as the primary cost driver.
Feature Engineering: Created an obese_smoker interaction feature that significantly boosted model accuracy.  
Hyperparameter Tuning: Used GridSearchCV to optimize the Random Forest architecture.  
Model Performance: Compared multiple algorithms (Linear, Polynomial, Decision Trees) to find the best fit.  
🛠️ Technologies Used: Python (Pandas, NumPy, Scikit-Learn). Visualization: Matplotlib, Seaborn. Notebook Environment: Google Colab 
Final Results Model R-Squared (R2)Baseline (Linear) 0.72 Decision Tree 0.81 Random Forest (Optimized)0.87
